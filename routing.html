IP Address, Packets, and Routing

This project is a continuation of a previous project titled,"History of the Internet".

While that project goes through a rudimentary introduction to the internet and its history, this project is focused on how the internet has evolved and works today.

The first topic covered in this project is the Domain Names and ICANN. Here we focus on what domains are and the regulatory body that manages them.

The next topic is how the addressing scheme of the internet as well has how information is sent and received. This is all covered in the IP Address, Packets, and Routing section.

How IP addresses and Domain Names correlate and allow users to get to where they want to go is covered in the next section, Domain Name Server (DNS).

Next we take a look at how the internet is accessed, as well as a brief discussion about security can be viewed in the HTTP and HTTPS protocols section.

Lastly, we discuss the visual evolution of the browser renderings through graphic markups thanks to W3C and HTML and CSS.
